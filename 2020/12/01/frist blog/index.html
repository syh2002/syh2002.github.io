#创建自己的GitHub 页面的手把手操作
#一. 搭建GitHub Pages
##0x01.注册一个GitHub账号（已注册的跳过这一步），登录页面。
##0x02.创建仓库。
###1.点击搜索栏下面的绿色按钮“NEW”。

###2.填写的仓库名为用户名.github.io（请再三查看，本人第一次制作时因为此原因重试了5遍还未发现错误)

##0x03启用Github pages
###1.点击“Settings”进入设置

###2.在下面找到“GitHub Pages”，显示如下，说明已开启。（一般创建完成后都会默认自动开启）

##0x04.选择主题
###1.点击“Choose a theme”选择主题

###2.选择自己喜欢的主题（后期可换），然后点击“select theme”

##0x05下载Git和node.js
###1.安装Node.js和配置好Node.js环境（务必最新版，否则都是泪），打开cmd命令行，成功页面如下

###2.安装Git和配置好Git环境，安装成功在电脑桌面上任何位置鼠标右键能够出现如下两个选择。

#二.HEXO
##0X01安装HEXO
###1.在自己认为合适的地方创建文件夹，然后在此文件夹下右击鼠标，选择“Git Bash here”

###2.输入 npm install -g hexo进行安装

###然后使用 hexo -v ,出现下面界面说明安装成功

###3.输入 hexo init 初始化文件夹（漫长的等待·······）

###看到最后的“Start blogging with Hexo！”，有没有鸡冻····

###4.输入 npm install,安装所需组件

###5.输入 hexo g ，第一次体验hexo（第一次就这样没了(╥╯^╰╥)）

###6.输入hexo s，启动本地服务器，通过访问 http://localhost:4000，正式体验Hexo（前戏结束，正式开始o(*￣︶￣*)o）

###成功了

##0X02更换主题
###1.安装主题（yilia
使用命令

hexo clean

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

###2、启动主题
###找到目录下的_config.yml 文件,打开找到 theme：属性，并设置为yilia(可用记事本打开)
###3、更新主题
cd themes/yilia

git pull

hexo g

hexo s

使用localhost:4000查看新主题！
